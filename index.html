CSS Styling - The Position Property

The position CSS property sets how an element is positioned in a document. The top, right, bottom, and left properties determine the final location of position elements. The following table summarizes how these different properties behave.

Property Element flow in document Description
static Normal Default value. Element position according to normal flow of document. top, right, bottom, left proprties have no effect
relative Positioned in normal flow of document, offset relative to itself The offset does not affect the position of any other elements, which therefore remain in their stated position.
absolute Removed from normal flow of document. No space created for the element in page layout. The element is positioned relative to its closest positioned ancestor (if any) or to the initial containing block. Its final position is determined by the values of top, right, bottom, and left.
fixed Removed from normal flow of document. The element is positioned relative to its initial containing block, which is the viewport in the case of visual media. Its final position is determined by the values of top, right, bottom, and left.
